
# Sistema Biblioteca Django

## Funciones
- CRUD libros
- CRUD préstamos
- Registrar devolución
- Consultar disponibilidad
- Reporte mensual

## Instalar Django
pip install django

## Migraciones
python manage.py makemigrations
python manage.py migrate

## Crear admin
python manage.py createsuperuser

## Ejecutar
python manage.py runserver
