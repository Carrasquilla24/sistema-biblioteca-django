Sistema de Biblioteca Django

Este proyecto consiste en un sistema de gestión de biblioteca desarrollado con Python y Django. Su objetivo es facilitar la administración de libros, usuarios y préstamos mediante una aplicación web sencilla y funcional.

El sistema permite registrar libros en la base de datos, consultar la información almacenada y realizar operaciones como editar o eliminar registros. Además, incluye funcionalidades para gestionar usuarios y controlar los préstamos y devoluciones de libros.

La aplicación utiliza Django como framework principal, junto con una base de datos SQLite para almacenar la información. También incorpora plantillas HTML y estilos CSS para la interfaz gráfica del sistema.

Entre las principales funcionalidades se encuentran:

* Registro y administración de libros.
* Gestión de usuarios.
* Control de préstamos y devoluciones.
* Panel de administración proporcionado por Django.
* Interfaz web accesible desde el navegador.
